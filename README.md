## How to Start the App

### 1. Clone repository

```bash
git clone <your-repo-url>
cd pac-man
```

### 2. Install dependencies

Install npm packages:

```bash
npm install
```
### 3. Run the project

```bash
npm run dev
```

---

The application should now be running locally.
